🔐 OAuth Fix Guide

Get your Google OAuth working in 3 simple steps

⚠️ The Problem

Your backend server is NOT running!

That's why you're seeing "Failed to fetch" and "Missing required parameter: client_id" errors.

1

Start Backend Server

Open a terminal and run:

START-BACKEND-NOW.bat
✅ Wait for this line:
OAuth configured: Client ID loaded (199412317897...)
Application startup complete

Keep this terminal window open!

2

Start Frontend Server

Open a NEW terminal and run:

START-FRONTEND-NOW.bat
✅ Wait for this line:
Local: http://localhost:5173/

Keep this terminal window open too!

3

Test OAuth Login

Open your browser and go to:

http://localhost:5173/login

Click "Continue with Google"

✅ You should see:
Google's consent screen (blue Google page)

🎉 That's It!

Once both servers are running, OAuth will work perfectly!

🐛 Troubleshooting

  • "Failed to fetch" → Backend is not running. Go to Step 1.
  • "Missing required parameter: client_id" → Check backend terminal. Should see "✅ OAuth configured"
  • Backend won't start → Make sure you're in the project root directory
  • Port already in use → Kill existing processes: netstat -ano | findstr :8000
